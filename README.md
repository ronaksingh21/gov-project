# AI Regulation in the United States

Interactive research website for an IDP project by Advaith V. & Ronak S.

**Live site:** [Enable GitHub Pages to deploy]

## What's inside

A single-page app with 6 sections:

| Page | Content |
|------|---------|
| Overview | Thesis, key stats, navigation |
| Advaith's argument | AI in hiring & criminal justice — 2 body paragraphs with evidence |
| Ronak's argument | Federal AI safety agency — 2 body paragraphs with evidence |
| Policy timeline | Key events from Nov 2023 to today |
| Global comparison | EU vs UK vs US vs China approaches |
| Interview email | Draft email to Professor Ryan Calo (UW Tech Policy Lab) |

## How to run locally

Just open `index.html` in a browser. No build step, no dependencies.

## How to deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under "Branch", select `main` and `/ (root)`
4. Click **Save**
5. Your site will be live at `https://<your-username>.github.io/<repo-name>`
